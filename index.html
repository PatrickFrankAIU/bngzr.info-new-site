About BNGZR

We are one of the top Unions in Trainstation 2. We were #1 in Fall 2025, and have placed at least one team (usually two, sometimes 3) in the Top 25 in every season the game has had so far.

We're especially good at helping new players learn how to earn a 100-slot train.

Requirements

There are no requirements for performance or dispatching. We have 5+ competitive teams, and new members start below the Top 50 and work their way up through SP production (Union jobs).

A report is posted each morning showing everyone's performance.

How to join?

If you'd like to learn more, please join our Discord server via the button below and post a message in our public channel. An officer will start the process with you as soon as possible.

Discord is required for all members, and we post announcements and data there every day.
